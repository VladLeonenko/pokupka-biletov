# Стек технологий PrimeCoder

## 🎯 Общая архитектура

Проект построен на современной архитектуре с разделением на frontend и backend, используя лучшие практики разработки.

---

## 🖥️ Frontend

### Основной стек
- **React 18.3.1** — библиотека для построения пользовательских интерфейсов
- **TypeScript 5.6.3** — типизированный JavaScript для надежности кода
- **Vite 5.4.10** — быстрый сборщик и dev-сервер
- **React Router 6.28.0** — маршрутизация в SPA

### UI/UX библиотеки
- **Material-UI (MUI) 6.1.6** — компонентная библиотека для современного дизайна
  - `@mui/material` — основные компоненты
  - `@mui/icons-material` — иконки
  - `@emotion/react` и `@emotion/styled` — стилизация
- **Framer Motion 12.23.24** — анимации и переходы

### Управление состоянием и данными
- **TanStack React Query 5.59.19** — управление серверным состоянием, кеширование, синхронизация
- **React Hook Form 7.53.0** — управление формами с валидацией
- **Zod 3.23.8** — валидация схем данных

### Дополнительные инструменты
- **Chart.js 4.4.4** + **react-chartjs-2 5.2.0** — визуализация данных и графики
- **DOMPurify 3.1.7** — санитизация HTML для безопасности
- **React Quill 2.0.0** — WYSIWYG редактор для контента
- **html2canvas 1.4.1** + **jspdf 3.0.4** — экспорт в PDF
- **date-fns 4.1.0** — работа с датами

### Сборка и разработка
- **Vite** — быстрая сборка и HMR (Hot Module Replacement)
- **TypeScript** — строгая типизация
- **@vitejs/plugin-react-swc** — оптимизированная компиляция React

---

## ⚙️ Backend

### Основной стек
- **Node.js** (ES Modules) — серверная платформа
- **Express 4.19.2** — веб-фреймворк для Node.js
- **PostgreSQL** — реляционная база данных
  - **pg 8.12.0** — драйвер для PostgreSQL

### Безопасность и аутентификация
- **Helmet 8.1.0** — защита HTTP-заголовков
- **CORS 2.8.5** — настройка Cross-Origin Resource Sharing
- **express-rate-limit 8.2.1** — ограничение частоты запросов
- **jsonwebtoken 9.0.2** — JWT токены для аутентификации
- **bcryptjs 3.0.2** — хеширование паролей

### Обработка данных
- **Multer 1.4.5-lts.1** — загрузка файлов
- **Sharp 0.34.5** — обработка и оптимизация изображений
- **Cheerio 1.0.0-rc.12** — парсинг HTML на сервере
- **csv-parser 3.2.0** — парсинг CSV файлов
- **Zod 4.1.13** — валидация данных на сервере

### Автоматизация и тестирование
- **Playwright 1.56.1** — автоматизация браузера, создание скриншотов, E2E тестирование

### Дополнительные сервисы
- **Nodemailer 6.9.15** — отправка email
- **Compression 1.7.4** — сжатие ответов сервера
- **dotenv 16.4.5** — управление переменными окружения

## 🗄️ База данных

- **PostgreSQL** — основная реляционная БД
  - Миграции через SQL файлы
  - JSONB для хранения структурированных данных
  - Индексы для оптимизации запросов

### Основные таблицы
- `pages` — страницы сайта
- `cases` — кейсы/портфолио
- `products` — товары/услуги
- `blog_posts` — статьи блога
- `carousels` — карусели и слайдеры
- `images` — метаданные изображений
- `users` — пользователи и аутентификация
- `orders` — заказы
- `tasks` — задачи и планировщик
- `clients` — клиенты
- `forms` — формы обратной связи

## 📦 Особенности архитектуры
