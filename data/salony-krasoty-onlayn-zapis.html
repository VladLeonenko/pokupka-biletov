День 1: выявили главную проблему

Статистика до запуска:

  • Пн: 3 записи (2 сброса, 1 дошла)
  • Вт: 2 записи (3 сброса)
  • Ср: 4 записи (1 дошла)

Вывод: Клиент ищет «салон Химки» → звонит → тишина → уходит к конкуренту.

Решение: Замена телефона на Telegram → услуга → мастер → время → оплата.

Схема работы системы

Клиент → «Хочу стрижку»

Telegram Bot → «Дата?» «15:00 ок?» «✅ Записано»

amoCRM → Сделка «Запись», статус OK

Мастер → «Клиент записан»

Код Telegram бота (Node.js):

const { Telegraf } = require('telegraf');

bot.command('start', async ctx => {
  const services = await amoCRM.getServices();
  ctx.reply('Выберите услугу:', servicesKeyboard(services));
});

bot.action('service_strizhka', async ctx => {
  const slots = await amoCRM.getFreeSlots(ctx.session.date);
  ctx.reply('Свободное время:', slotsKeyboard(slots));
});

День 14: подключили предоплату

Проблема: 23% клиентов не приходили.

Решение: Яндекс.Касса, предоплата 30%.

bot.action('pay_1500', async ctx => {
  const payment = await yandexKassa.createPayment({
    amount: { value: '1500.00', currency: 'RUB' },
    receipt: { phone: ctx.session.phone }
  });
  ctx.reply('Оплатите для подтверждения:', payment.confirmationUrl);
});

Отмены: 23% → 4%.
