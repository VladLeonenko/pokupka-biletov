# 🚀 Быстрые команды для работы с сервером

## Выход из различных программ

### PostgreSQL (psql)

Если вы в psql и видите приглашение типа `primecoder_prod=#` или `postgres=#`:

```bash
# Способ 1 (рекомендуется):
\q

# Способ 2:
\quit

# Способ 3:
exit

# Способ 4:
Ctrl+D
```

**Другие полезные команды psql:**
- `\l` - показать все базы данных
- `\c database_name` - подключиться к базе данных
- `\dt` - показать все таблицы
- `\du` - показать всех пользователей
- `\?` - справка по командам psql
- `\h` - справка по SQL командам

---

### Nano (текстовый редактор)

Если вы редактируете файл в nano (видите внизу `^X Exit`, `^O Write Out`):

```bash
# Сохранить и выйти:
Ctrl+O  # Write Out (сохранить)
Enter   # подтвердить имя файла
Ctrl+X  # Exit (выйти)

# Выйти без сохранения:
Ctrl+X  # если есть изменения, нажмите N (No)
```

**Горячие клавиши nano:**
- `Ctrl+O` - сохранить файл
- `Ctrl+X` - выйти
- `Ctrl+K` - вырезать строку
- `Ctrl+U` - вставить строку
- `Ctrl+W` - поиск
- `Ctrl+\` - поиск и замена
- `Ctrl+G` - справка

---

### Vim/Vi (текстовый редактор)

Если вы в vim (видите `-- INSERT --` внизу или просто курсор):

```bash
# Выйти из режима редактирования:
Esc

# Затем:
:wq    # сохранить и выйти (write and quit)
:w     # только сохранить (write)
:q     # только выйти (quit)
:q!    # выйти без сохранения (force quit)
```

**Режимы vim:**
- `i` - войти в режим редактирования (INSERT)
- `Esc` - выйти из режима редактирования
- `:w` - сохранить
- `:q` - выйти
- `:wq` - сохранить и выйти

---

### Less/More (просмотр файлов)

Если вы просматриваете файл через `less` или `more`:

```bash
q       # выйти (quit)
Space   # следующая страница
b       # предыдущая страница (только в less)
/текст  # поиск (только в less)
```

---

### Top/Htop (мониторинг процессов)

```bash
q       # выйти (quit)
Ctrl+C  # также выйти
```

---

### SSH сессия

```bash
exit    # выйти из SSH
logout  # альтернатива
Ctrl+D  # также выйти
```

## Частые ситуации

### "Я в psql и не знаю как выйти"
```bash
\q
# или
Ctrl+D
```

### "Я редактирую файл в nano и не знаю как сохранить"
```bash
Ctrl+O  # сохранить
Enter   # подтвердить
Ctrl+X  # выйти
```

### "Я случайно открыл vim и не знаю что делать"
```bash
Esc     # выйти из режима редактирования
:q!     # выйти без сохранения
Enter   # подтвердить
```

### "Я в какой-то программе и не знаю как выйти"
Попробуйте по порядку:
1. `q` или `Q`
2. `exit`
3. `Ctrl+C`
4. `Ctrl+D`
5. `Esc` затем `:q` (если это vim)

## Полезные команды для работы с PostgreSQL

### Подключение к базе данных
```bash
# Подключиться как пользователь postgres
sudo -u postgres psql

# Подключиться к конкретной базе
sudo -u postgres psql -d primecoder_prod

# Подключиться как конкретный пользователь
psql -U primecoder_user -d primecoder_prod

# С паролем через переменную окружения
PGPASSWORD='пароль' psql -U primecoder_user -d primecoder_prod
```

### Работа с базой данных
```sql
-- Показать все базы данных
\l

-- Подключиться к базе
\c primecoder_prod

-- Показать все таблицы
\dt

-- Показать структуру таблицы
\d table_name

-- Выполнить SQL команду
SELECT * FROM users LIMIT 5;

-- Выйти
\q
```

---

**Совет:** Если застряли в любой программе, попробуйте `Ctrl+C` или `q` - это работает в большинстве случаев!
