# Скрипт для создания скриншотов кейсов

Этот скрипт автоматически создает скриншоты сайтов из `donor_url` кейсов и обрабатывает их для использования в качестве изображений кейсов.

## Возможности

✅ **Автоматические скриншоты** - делает скриншоты сайтов через Playwright  
✅ **Обработка изображений** - автоматически ресайзит и обрезает до нужных размеров  
✅ **Множественные изображения** - создает Hero (1920x1080) и 2 изображения для галереи (1200x800)  
✅ **Разные части страницы** - делает скриншоты разных секций для разнообразия  
✅ **Обновление БД** - автоматически обновляет `hero_image_url` и `gallery` в базе данных  

## Установка

1. Установите зависимости:
```bash
cd backend
npm install
```

2. Установите браузеры для Playwright:
```bash
npx playwright install chromium
```

## Использование

### Базовое использование (только кейсы без изображений)
```bash
npm run screenshot:cases
```

### Обновить все кейсы (включая те, у которых уже есть изображения)
```bash
node scripts/screenshot-case-sites.js --force
# или
node scripts/screenshot-case-sites.js -f
```

### Пропустить кейсы с существующими изображениями
```bash
node scripts/screenshot-case-sites.js --skip-existing
# или
node scripts/screenshot-case-sites.js -s
```

## Что делает скрипт

1. **Получает кейсы** из БД с `donor_url`
2. **Открывает сайт** в браузере (headless режим)
3. **Ждет загрузки** контента (включая ленивые изображения)
4. **Делает скриншоты**:
   - Hero: верх страницы (1920x1080)
   - Gallery 1: средняя часть страницы (1200x800)
   - Gallery 2: нижняя часть страницы (1200x800)
5. **Обрабатывает изображения**:
   - Ресайз с сохранением пропорций
   - Обрезка до нужных размеров
   - Сохранение в PNG формате
6. **Обновляет БД** с путями к изображениям

## Технические детали

- **Формат**: PNG
- **Hero размер**: 1920x1080px (16:9)
- **Gallery размер**: 1200x800px (3:2)
- **Качество**: 90%
- **Таймаут**: 30 секунд на страницу
- **Задержка**: 2 секунды между кейсами, 1 секунда между скриншотами

## Требования

- Node.js 18+
- PostgreSQL база данных с таблицей `cases`
- Playwright с установленным Chromium
- Sharp для обработки изображений

## Обработка ошибок

Скрипт продолжает работу даже при ошибках на отдельных кейсах:
- ❌ Ошибки логируются в консоль
- ✅ Успешно обработанные кейсы обновляются в БД
- 📊 В конце выводится статистика: обработано/пропущено/ошибок
