# Восстановление данных товаров PrimeCoder

## Ситуация

Данные товаров (описания, картинки, FAQ) были созданы через админ-панель, но потеряны при загрузке данных amani/umagazine в БД.

## Варианты восстановления

### Вариант 1: Восстановление из скриншотов/документов

Если у тебя есть:
- Скриншоты страниц товаров
- Документы с описаниями
- Заметки с FAQ

Можно восстановить вручную через админ-панель.

### Вариант 2: Создание заново через админ-панель

1. Зайди в админ-панель: `http://localhost:5173/products` (или на боевом сервере)
2. Открой каждый товар для редактирования
3. Заполни:
   - Полное описание (`full_description_html`)
   - Картинки (`image_url`, `gallery`)
   - FAQ в `content_json.faq`
   - SEO метаданные

### Вариант 3: Использование шаблонов

Можно использовать структуру `content_json` из типов:

```typescript
{
  header: { title, description, buttons },
  description: { title, text },
  priceSection: { title, tariffs: [...] },
  workSteps: { title, description, steps: [...] },
  stats: { title, description, items: [...] },
  team: { title, description, members: [...] },
  relatedServices: { title, services: [...] },
  subscribe: { items: [...] },
  backgroundImages: { grayBgUrl, yellowBgUrl },
  faq: {
    title: "Часто задаваемые вопросы",
    items: [
      { question: "...", answer: "..." },
      ...
    ]
  }
}
```

### Вариант 4: Проверка других источников

Проверь:
- Другие репозитории/проекты
- Бэкапы на других серверах
- Документы/файлы с описаниями услуг
- История браузера (если данные были в localStorage)
