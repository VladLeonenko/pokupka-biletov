# Простая инструкция по настройке прокси для OpenAI

## Проблема
OpenAI API недоступен в вашем регионе (ошибка 403 "unsupported_country_region_territory")

## Решение: Использовать прокси

### Быстрый вариант (рекомендуется): Cloudflare Workers

**Преимущества:**
- ✅ Бесплатно
- ✅ Быстро настраивается (5 минут)
- ✅ Надежно
- ✅ Не требует установки дополнительного ПО

**Инструкция:** См. файл `CLOUDFLARE_PROXY_SETUP.md`

### Альтернативные варианты прокси

#### 1. Готовые прокси-сервисы (платные, но простые)

Некоторые сервисы предлагают готовые прокси для OpenAI:
- Поищите в Google: "OpenAI API proxy service"
- Обычно стоят $5-20/месяц

#### 2. Настройка через VPN на сервере

Если у вас есть VPS с VPN:
1. Подключите сервер к VPN с доступом к OpenAI
2. Используйте прокси-сервер на этом VPS

#### 3. Использование альтернативных AI API

Можно использовать другие AI сервисы вместо OpenAI:
- Anthropic Claude API
- Google Gemini API
- Yandex GPT (для России)

## Как добавить прокси URL

После того, как получите прокси URL:

1. Откройте файл `backend/.env`
2. Добавьте строку:
   ```env
   OPENAI_PROXY_URL=https://ваш-прокси-url.com
   ```
3. Перезапустите backend:
   ```bash
   cd backend
   lsof -ti:3000 | xargs kill -9
   node app.js
   ```

## Проверка

Запустите тест:
```bash
cd backend
node scripts/testOpenAIProxy.js
```

Если видите "✅ Подключение успешно!" - все работает!

## Нужна помощь?

Если возникли проблемы:
1. Проверьте логи backend - там будут подробные сообщения об ошибках
2. Убедитесь, что прокси URL правильный
3. Проверьте, что прокси доступен из вашей сети
