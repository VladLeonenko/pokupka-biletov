# Обновление API ключа OpenAI

Если вы получаете ошибку "Incorrect API key provided" или "401 Unauthorized", значит нужно обновить API ключ.

## Шаг 1: Получите новый API ключ

1. Зайдите на https://platform.openai.com/account/api-keys
2. Войдите в свой аккаунт OpenAI
3. Убедитесь, что на счету есть средства
4. Создайте новый API ключ:
   - Нажмите "Create new secret key"
   - Дайте ключу имя (например: "CMS Production")
   - Скопируйте ключ (он показывается только один раз!)

## Шаг 2: Обновите ключ в .env

1. Откройте файл `backend/.env`
2. Найдите строку с `OPENAI_API_KEY`
3. Замените старый ключ на новый:

```env
OPENAI_API_KEY=sk-ваш-новый-ключ-здесь
```

**ВАЖНО:**
- Ключ должен начинаться с `sk-` или `sk_`
- Скопируйте ключ полностью (обычно около 50 символов)
- Не добавляйте пробелы или переносы строк

## Шаг 3: Перезапустите backend

Выполните в терминале:

```bash
cd backend
lsof -ti:3000 | xargs kill -9
node app.js
```

Или используйте команду из инструкции.

## Шаг 4: Проверьте работу

```bash
cd backend
node scripts/testOpenAIProxy.js
```

Должно быть: ✅ Подключение успешно!

## Если все еще не работает

1. Проверьте, что ключ скопирован полностью (нет обрезанных символов)
2. Убедитесь, что на счету OpenAI есть средства
3. Проверьте, что ключ не истек (создайте новый)
4. Убедитесь, что backend перезапущен после изменения .env
5. Проверьте логи: `tail -50 /tmp/backend.log`

## Безопасность

⚠️ Никогда не коммитьте `.env` файл в Git!
- Файл `.env` уже должен быть в `.gitignore`
- API ключ должен быть только у вас на сервере
