# Письма для UniSender: сценарии и тексты

Ниже — **4 сценария** отправки и готовые тексты. Создай в UniSender по одному шаблону на сценарий и подставь переменные. В UniSender используется синтаксис **`[%имя_переменной%]`**.

---

## Сценарии отправки

| Сценарий | Когда срабатывает | Шаблон в коде |
|----------|-------------------|----------------|
| **High potential** | AI-аудит дал высокий потенциал (business_potential = high) | high_potential |
| **Medium potential** | Средний потенциал (medium) | medium_potential |
| **Low potential** | Низкий потенциал (low) | low_potential |
| **Без аудита (базовый)** | Личная почта (gmail/yandex/mail.ru) или нет сайта | basic_no_audit |

Канал: если с сайта найден телефон — сначала **UniSender Telegram**, при ошибке — **Email**. Иначе — только **Email**.

---

## Переменные для подстановки

Используй в шаблонах UniSender так: `[%company_name%]`, `[%website%]` и т.д.

| Переменная | Описание | Где есть |
|------------|----------|----------|
| `[%company_name%]` | Название компании | все |
| `[%website%]` | URL сайта | с аудитом |
| `[%audit_score%]` | Оценка 0–100 | с аудитом |
| `[%audit_summary%]` | Краткое резюме аудита (текст) | с аудитом |
| `[%personalized_intro%]` | Персонализированное вступление от AI | с аудитом |
| `[%recommended_approach%]` | Рекомендованный подход от AI | с аудитом |
| `[%conversion_boost%]` | Пример прироста конверсии (число %) | high |
| `[%ux_improvement%]` | Пример улучшения UX (число %) | high |

Сейчас бэкенд **не** использует template_id UniSender: он сам подставляет эти данные в текст и отправляет через sendEmail/sendTelegram. Ты можешь создать в UniSender такие же письма для единого стиля и рассылок вручную; при необходимости потом можно добавить отправку по template_id.

---

## 1. High potential (высокий потенциал)

**Тема письма:**  
`Эксклюзивное предложение для [%company_name%]`

**Текст письма (тело):**

```
Здравствуйте!

Мы провели экспертный анализ вашего сайта [%website%] и обнаружили значительный потенциал для роста.

📊 Общая оценка: [%audit_score%]/100
🎯 Потенциал роста: Высокий

[%personalized_intro%]

[%audit_summary%]

✨ Что можно улучшить:
• Современный дизайн и UX → рост конверсии
• Интеграция AI-инструментов → автоматизация процессов
• Оптимизация под современные тренды

Готовы обсудить, как мы можем помочь вам реализовать этот потенциал? Давайте назначим встречу.

С уважением,
[Ваша подпись]
```

## Рекомендации по оформлению в UniSender

- В визуальном редакторе замени плейсхолдеры на переменные в формате `[%variable_name%]`.
- Укажи отправителя (SENDER_EMAIL, SENDER_NAME) в настройках рассылки или в .env бэкенда.
- Один раз проверь отправку тестового письма себе, чтобы убедиться, что переменные подставляются.
- Для Telegram (если используешь тот же текст) — сократи при необходимости до 1–2 абзацев.

После создания шаблонов сохрани их ID — если захочешь, чтобы бэкенд отправлял по template_id, можно будет добавить это в код.
