ПРОБЛЕМА: OpenAI API недоступен в вашем регионе
РЕШЕНИЕ: Настроить прокси через Cloudflare Workers (5 минут)

═══════════════════════════════════════════════════════════════
ШАГ 1: Создайте Cloudflare Worker
═══════════════════════════════════════════════════════════════

1. Зайдите на https://workers.cloudflare.com
2. Войдите или зарегистрируйтесь (бесплатно)
3. Нажмите "Create a Worker"
4. Вставьте код из файла: CLOUDFLARE_PROXY_SETUP.md
5. Сохраните и деплойте

═══════════════════════════════════════════════════════════════
ШАГ 2: Получите URL Worker
═══════════════════════════════════════════════════════════════

После деплоя вы получите URL вида:
https://your-worker.workers.dev

Скопируйте этот URL

═══════════════════════════════════════════════════════════════
ШАГ 3: Добавьте в .env
═══════════════════════════════════════════════════════════════

Откройте backend/.env и добавьте:

OPENAI_PROXY_URL=https://your-worker.workers.dev

(БЕЗ /v1/chat/completions - добавятся автоматически)

═══════════════════════════════════════════════════════════════
ШАГ 4: Перезапустите backend
═══════════════════════════════════════════════════════════════

cd backend
lsof -ti:3000 | xargs kill -9
node app.js

═══════════════════════════════════════════════════════════════
ШАГ 5: Проверьте
═══════════════════════════════════════════════════════════════

cd backend
node scripts/testOpenAIProxy.js

Должно быть: ✅ Подключение успешно!
