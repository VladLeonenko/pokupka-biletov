# HTML-шаблоны писем пайплайна (Sales Pipeline)

Адаптивные HTML-письма в **стилистике сайта PrimeCoder**: тёмный фон (#141414, #1d1d1d), акцент #ffbb00, шрифты Arial/Helvetica, табличная вёрстка для совместимости с почтовыми клиентами.

## Структура каждого письма

1. **Шапка** — логотип (ссылка на prime-coder.ru). Крупная типографика (H1 28px, секции 22px).
2. **Основной блок** — персонализированный текст (аудит / приветствие), стилистика сайта (#141414, #ffbb00).
3. **Блок «Почему мы» (выгоды с сайта)** — 73% клиентов рост трафика 200%+ за 6 мес, 150+ проектов, 8 лет; плюс прозрачность, отчётность, поддержка, экономия до 40%, окупаемость от 3 мес.
4. **Блок выгоды (калькулятор)** — «Экономия до 40% против рынка · Окупаемость от 3 месяцев · Прирост прибыли в год» + кнопка «Рассчитать на сайте» → `/tools/roi-calculator`.
5. **Блок «Этапы работы»** — 7 этапов: Анализ и планирование → Проектирование и UX → Дизайн → Разработка → Контент и SEO → Тестирование → Запуск и продвижение (как на сайте).
6. **Три карточки услуг с обложками** — у каждой карточки изображение (uniq-adv.png, complex-adv.png, coast-adv.png с prime-coder.ru/legacy/img/). Сайты под ключ, SEO, AI и автоматизация. Каждая ведёт в каталог.
7. **Быстрые ссылки** — Каталог · Кейсы · Калькулятор ROI · Контакты.
8. **CTA-кнопка** — «Связаться с нами» / «Записаться на встречу» и т.д. → `/#contacts`.
9. **Подвал** — контакты, **кнопка отписки** `[%unsubscribe_url%]`.

**Анимации:** появление блоков (fadeInUp) с разным animation-delay (0s, 0.12s, 0.24s …) — при открытии письма блоки поочерёдно «въезжают». В Outlook анимации отключены через `[if mso]`.

## Файлы и переменные

| Файл | Сценарий | Переменные UniSender |
|------|----------|----------------------|
| `high-potential.html` | Высокий потенциал (аудит) | company_name, website, audit_score, personalized_intro, audit_summary, unsubscribe_url |
| `medium-potential.html` | Средний потенциал | company_name, website, audit_score, audit_summary, unsubscribe_url |
| `low-potential.html` | Низкий потенциал | company_name, website, audit_summary, unsubscribe_url |
| `basic-no-audit.html` | Без аудита (личная почта) | unsubscribe_url |

## Особенности

- **Стилистика сайта:** #141414 фон, #ffbb00 акцент, #1d1d1d карточки, #d0d0d0/#888 текст. Без сторонних шрифтов — только Arial/Helvetica для стабильной доставки и отображения.
- **Выгода из калькулятора:** текст блока выгоды — это интерпретация типичных результатов ROI-калькулятора (экономия %, окупаемость, прирост прибыли). Ссылка ведёт на `/tools/roi-calculator`.
- **Три карточки услуг с обложками (low-potential):** Сайты под ключ — `react-2-1770584437676.png`, SEO продвижение — `SEO---1770911540508.png`, AI и автоматизация — `AI_boost_team-2-1770909008448.png` (все в `https://prime-coder.ru/uploads/images/`). Карточки целиком кликабельны → каталог.
- **low-potential: шапка и фон.** Шапка — баннер (`email-header-banner.png`), местами фон — сфера (`email-sphere-bg.png`). Файлы лежат в `backend/email-templates/sales/assets/`. Для отображения в письме загрузите их в `uploads/images/` на сервере (или укажите в шаблоне свои URL).
- **low-potential: блок кейсов.** Ряд из 3 карточек (Дома России, Полигон, YOUTRAVEL) + ссылка «Смотреть все кейсы» → `/portfolio`. В письме карусель реализована как статичный ряд карточек (без JS).
- **Быстрые ссылки:** Каталог, Кейсы, Калькулятор ROI, Контакты — всегда в одном месте для быстрого перехода.
- **Отписка:** в подвале явная ссылка «Отписаться от рассылки» с плейсхолдером `[%unsubscribe_url%]` (UniSender подставит сам).
- **Анимация:** только лёгкий `fadeIn` (0.4s) на основном блоке; многие клиенты анимации не поддерживают — письмо остаётся читаемым без них.
- **Адаптивность:** `@media (max-width: 600px)` — одна колонка, карточки стекаются, кнопки на всю ширину.

## Использование в UniSender

1. Создай 4 шаблона в UniSender (по одному на сценарий).
2. В каждом шаблоне переключись в режим HTML и вставь содержимое соответствующего `.html` файла.
3. Переменные уже в формате `[%company_name%]`, `[%website%]` и т.д. — UniSender подставит их при отправке.
4. Укажи тему письма (Subject) — см. `backend/SALES_PIPELINE_EMAIL_TEMPLATES.md`.
5. Preheader уже есть в каждом письме (скрытый div в начале body) — при желании измени текст под сценарий.
