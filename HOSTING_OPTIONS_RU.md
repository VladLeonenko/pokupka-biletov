## 📝 ИНСТРУКЦИЯ ПО РАЗМЕЩЕНИЮ НА YANDEX CLOUD

### Шаг 1: Регистрация
1. Перейдите на https://cloud.yandex.ru/
2. Зарегистрируйтесь через Яндекс ID
3. Создайте платежный аккаунт (можно без карты для теста)

### Шаг 2: Создание инфраструктуры

**Вариант A: Docker контейнеры (рекомендуется)**
```bash
# Создать Dockerfile для backend
# Создать docker-compose.yml
# Задеплоить через Yandex Container Registry
```

**Вариант B: Compute Cloud (VM)**
1. Создать виртуальную машину (Ubuntu 22.04)
2. Установить Node.js, PostgreSQL
3. Настроить Nginx как reverse proxy
4. Задеплоить приложение

### Шаг 3: База данных
1. Создать Yandex Managed PostgreSQL
2. Импортировать данные
3. Настроить подключение из приложения

### Шаг 4: Frontend
1. Собрать React приложение (`npm run build`)
2. Загрузить в Yandex Object Storage
3. Настроить Cloud CDN для ускорения

### Шаг 5: Доменное имя
1. Купить домен (например, на reg.ru или timeweb.com)
2. Настроить DNS записи
3. Подключить SSL сертификат (Let's Encrypt бесплатно)
